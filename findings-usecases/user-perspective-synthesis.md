# User Perspective — Condensed

**Real community feedback. "The minimal one that does not get in your way."**

## What Users Actually Value

| Feature | Why |
|---------|-----|
| Minimal core (4 tools) | Start clean, add only what you need |
| Extension API | "No forks, no patches" |
| Hot reload (`/reload`) | Iterate without restarting |
| Model choice (20+ providers) | Use whatever works |

## Top 3 Extensions Users Install

1. **Memory** — without it, Pi starts fresh every session
2. **MCP adapter** — reuse tools from other agents
3. **Sub-agents** — parallel work in isolated agents

## Ecosystem Stats

200+ npm packages, 60+ community skills, 76 themes. pi-subagents is most popular. LazyPi provides curated starter pack. Rust port (pi_agent_rust, 823★) proves embedded JS runtimes can run TypeScript extensions from compiled Rust binaries.

## Key Takeaways

- Extensions are the primary customization surface
- Hot reload is table-stakes
- Memory is the #1 requested feature
- Minimal core is praised, not criticized
- Rust+JS/Wasm for extensions works (proven by Rust port)
- Third-party curation emerges naturally (LazyPi, awesome lists)
